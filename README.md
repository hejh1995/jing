# jing微信小程序开发实战。
## 组件化开发
## 主要实现的功能：
- 请求的封装：封装一个类具有request方法，内部调用wx.request方法，返回一个promise，可以避免回调地狱。
- 组件： 喜欢组件，页面组件，加载信息组件，
- 页面切换。如果是音乐页面，在组件的attached阶段，判断音乐是否播放，旋转背景，更换图标。
- 音乐播放：wx.getBackgroundAudioManager()，调取微信背景音乐播放，有onplay，onpause，onstop，onended方法。
